# gil-web.com

**Gino Ladowitch**'s Portfolio website

URL: [gil-web.com/](https://gil-web.com/)

Made with:

- Zola
- Sass

Featuring:

- SVG animations

## Installation

- Install [Zola](https://www.getzola.org/) on your system

At project root:

```sh
zola serve
```

Zola will handle sass compilation.
